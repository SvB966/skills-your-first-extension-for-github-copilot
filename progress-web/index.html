Student Progress Tracker

This is a minimal API-driven service. Use API endpoints to manage data.

  • GET /api/students - list students
  • POST /api/students - add student {"name": "Alice"}
  • GET /api/teachers - list teachers
  • POST /api/teachers - add teacher {"name": "Mr. Smith"}
  • GET /api/courses - list courses
  • POST /api/courses - add course {"name":"Math","teacher_id":1,"year":2025}
  • GET /api/grades - list grades
  • POST /api/grades - add grade {"student_id":1,"course_id":1,"grade":"A"}
